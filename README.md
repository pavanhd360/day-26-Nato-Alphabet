This project is a NATO Alphabet program.

It is a basic illustration of list comprehension and dictionary usage.
In this program, we first import the pandas library. 
Then, we read data from a file named NATO_phonetic_alphabet.csv 
using pandas and load it into the main.py file.

The data is used to create a dictionary where each letter corresponds 
to its NATO phonetic alphabet code.

The user is asked to enter a word. Each letter of the entered word is then converted
into its corresponding NATO phonetic alphabet word using dictionary and list comprehension.
